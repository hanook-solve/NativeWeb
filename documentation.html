📖 Introduction

NativeWeb is a production-ready Android WebView template that converts any website into a fully native Android app. With just one line of code changed, your website becomes a powerful Android app with push notifications, ads, offline support, and much more.

✅ One Line Setup: Change START_URL in Config.java and you're done!

🚀 Quick Start

Get your app running in just 3 steps!

1

Set Your Website URL

Open Config.java and change one line:

public static final String START_URL = "https://www.yourwebsite.com";
2

Add Firebase (for push notifications)

Go to console.firebase.google.com, create a project, register your app and download google-services.json into the app/ folder.

3

Add AdMob (for ads)

Go to admob.google.com, create your app and replace the ad unit IDs in Config.java.

✅ That's it! Build and run your app!

📦 Installation

1

Extract ZIP File

Extract the downloaded ZIP to a folder on your PC.

2

Open In Android Studio

Launch Android Studio → Click Open → Select the NativeWeb folder → Click OK

3

Add gradle.properties

Create gradle.properties in root folder with:

android.useAndroidX=true
android.enableJetifier=true
org.gradle.jvmargs=-Xmx4096m
org.gradle.parallel=true
org.gradle.caching=true
4

Sync Gradle

Wait for Gradle sync to complete. Click File → Sync Project with Gradle Files

5

Run The App

Connect your Android device or start an emulator. Click the green ▶ Run button!

⚙️ Configuration

All settings are in one file: app/src/main/java/com/webviewapp/Config.java

Website Settings

// Your website URL — CHANGE THIS!
public static final String START_URL = "https://www.yourwebsite.com";

// Keep navigation inside this domain
// Set to "" to allow all URLs
public static final String ALLOWED_DOMAIN = "yourwebsite.com";

App Identity

public static final String APP_NAME     = "NativeWeb";
public static final String COMPANY_NAME = "Your Company";
public static final String PLAY_STORE_URL = "https://play.google.com/...";

AdMob Settings

// Set to false to disable all ads
public static final boolean ADS_ENABLED = true;

// Your AdMob ad unit IDs
public static final String ADMOB_BANNER_ID      = "ca-app-pub-xxx/xxx";
public static final String ADMOB_INTERSTITIAL_ID = "ca-app-pub-xxx/xxx";

// Show interstitial every N page loads (0 = disabled)
public static final int INTERSTITIAL_EVERY_N_PAGES = 5;

Firebase Settings

// All devices subscribe to this topic automatically
public static final String FCM_DEFAULT_TOPIC = "all_users";

Splash Screen

// Minimum splash duration in milliseconds
public static final long SPLASH_DURATION_MS = 3000;

🎨 Branding & Customization

App Name

Change in res/values/strings.xml:

<string name="app_name">Your App Name</string>
<string name="splash_tagline">Your tagline here</string>

Brand Colors

Change in res/values/colors.xml:

<color name="colorPrimary">#2D1B69</color>    <!-- Main color -->
<color name="colorSecondary">#FFD700</color>  <!-- Accent color -->
<color name="splashBackground">#1A0533</color> <!-- Splash bg -->

App Icon

In Android Studio: Right click res → New → Image Asset → Upload your logo → Click Finish

Package Name

Change in app/build.gradle:

defaultConfig {
    applicationId "com.yourcompany.yourapp"
}

🔥 Firebase Setup

1

Create Firebase Project

Go to console.firebase.google.com → Click Add project → Follow the steps

2

Add Android App

Click Add app → Android → Enter your package name → Click Register app

3

Download google-services.json

Download the file and place it in the app/ folder

4

Enable Messaging

In Firebase Console → Click Messaging → Click Get Started

⚠️ Important: Package name in Firebase must exactly match your app's applicationId in build.gradle!

Send Push Notification

// Notification payload format
{
  "notification": {

    "body": "Your message here"
  },
  "data": {
    "url": "https://yourwebsite.com/page"  // optional deep link
  },
  "to": "/topics/all_users"
}

💰 AdMob Setup

1

Create AdMob Account

Go to admob.google.com → Sign in with Google → Create account

2

Add Your App

Click Apps → Add app → Android → Enter app name → Get App ID

3

Create Ad Units

Create Banner and Interstitial ad units → Copy the IDs

4

Update Config.java

public static final String ADMOB_BANNER_ID =
    "ca-app-pub-XXXXXXXX/XXXXXXXXXX";
public static final String ADMOB_INTERSTITIAL_ID =
    "ca-app-pub-XXXXXXXX/XXXXXXXXXX";
5

Update AndroidManifest.xml

<meta-data
    android:name="com.google.android.gms.ads.APPLICATION_ID"
    android:value="ca-app-pub-XXXXXXXXXXXXXXXX~XXXXXXXXXX"/>
⚠️ Never use test IDs in production! Your AdMob account may be suspended.

Disable Ads

// Set to false to completely disable all ads
public static final boolean ADS_ENABLED = false;

🌉 JavaScript Bridge

Call native Android functions directly from your website's JavaScript!

Available Methods

MethodDescriptionExample
showToast(msg)Show native toast Android.showToast("Hello!")
shareText(title, text)Open share sheet Android.shareText("Title", "Text")
openDialer(number)Open phone dialer Android.openDialer("+1234567890")
openEmail(to, sub, body)Open email client Android.openEmail("a@b.com", "Sub", "Body")
openBrowser(url)Open external browser Android.openBrowser("https://...")
openMaps(query)Open Google Maps Android.openMaps("Eiffel Tower")
isNetworkAvailable()Check connectivity Android.isNetworkAvailable()
getPackageName()Get app package name Android.getPackageName()
getAppVersion()Get app version Android.getAppVersion()
closeApp()Close the app Android.closeApp()

Example Usage In Your Website

// Check if running in NativeWeb app
if (typeof Android !== 'undefined') {
    // Running inside the app
    Android.showToast("Welcome to our app!");
} else {
    // Running in regular browser
    console.log("Running in browser");
}

// Show contact button only in app
document.getElementById('call-btn').onclick = function() {
    Android.openDialer("+1-800-123-4567");
};

// Share current page
document.getElementById('share-btn').onclick = function() {
    Android.shareText(
        document.title,
        window.location.href
    );
};

🔔 Push Notifications

How It Works

All devices automatically subscribe to the all_users topic when the app is first launched. Send notifications from Firebase Console or your server.

Send From Firebase Console

1

Go to Firebase Console → Messaging

2

Create new campaign

Select Firebase Notification messages

3

Set Target

Select Topic → all_users

4

Add Deep Link (optional)

In Additional Options → Custom data:

Key:   url
Value: https://yourwebsite.com/page

Change Default Topic

In Config.java:

public static final String FCM_DEFAULT_TOPIC = "your_topic_name";

📁 File Upload / Download

File Upload

File upload works automatically with any HTML <input type="file"> element. Users can choose from gallery or take a photo.

File Download

All file types are supported for download including PDF, ZIP, images, documents, videos and more. Downloads appear in the phone's notification bar and are saved to the Downloads folder.

Long Press Image Download

Users can long press any image on the website to get a download prompt.

❓ FAQ

How do I change the website URL?

Open Config.java and change START_URL to your website URL.

How do I disable ads?

Set ADS_ENABLED = false in Config.java.

App shows offline page even with internet?

Make sure ALLOWED_DOMAIN matches your website domain exactly. Set it to "" to allow all domains.

Push notifications not working?

Make sure the package name in Firebase matches exactly with applicationId in build.gradle.

How do I change the app colors?

Edit res/values/colors.xml and change colorPrimary and colorSecondary.

Can I use this for multiple websites?

Yes! Just change START_URL in Config.java for each app.

Links open in external browser?

Set ALLOWED_DOMAIN to match your website domain or set it to "" to keep all links inside the app.
